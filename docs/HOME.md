## 规则速查

### 补仓触发线（较近期高点下跌）

| 资产 | 1档 | 2档 | 3档 |
|------|------|------|------|
| 宽基类 | -8% | -15% | -25% |
| 科技主题类 | -10% | -20% | -30% |
| 标普500 | -10% | -15% | -20% |
| 纳斯达克科技 | -12% | -20% | -30% |
| 中概互联 | -15% | -25% | -35% |
| 黄金 | -12% | -20% | -30% |
| 债券 | 不设补仓线 | - | - |

### 止盈触发线（较补仓点上涨）

| 资产 | 1档 | 2档 | 3档 |
|------|------|------|------|
| 宽基类 | +15% | +30% | +50% |
| 科技主题类 | +20% | +40% | +60% |
| 标普500 | +15% | +25% | +35% |
| 纳斯达克科技 | +20% | +35% | +50% |
| 中概互联 | +25% | +40% | +60% |
| 黄金 | +20% | +35% | - |
| 债券 | 仓位占比 >30% 时减 | - | - |

### 特殊规则

- **债券弹药库**：其他资产触发第 3 档补仓时，卖出 10%~20% 债券仓位用于补仓
- **债券止盈**：债券仓位占比超过 30% 时，卖出超出部分
