# 产品说明

## 定位

面向个人投资者的基金顾投 AI 助手，通过自然语言交互实现持仓分析、基金诊断和组合管理。

## 核心场景

### 查看持仓

获取各基金基本信息 + 最新净值 + 近期收益，用表格展示数据。

### 分析持仓

1. **持仓概览** — 各基金基本信息 + 最新净值 + 近期收益
2. **资产配置** — 穿透后的股/债/现金比例
3. **行业分布** — 前 5 大行业集中度
4. **相关性分析** — 基金间重合度（分散是否充分）
5. **风险评估** — 组合整体风险等级
6. **诊断建议** — 集中度、风格偏移等问题提示
7. **策略评估** — 结合操作策略，判断是否触发补仓/止盈/纪律预警

数据来源：且慢 MCP（DiagnoseFundPortfolio / GetFundAssetClassAnalysis / GetFundsCorrelation / AnalyzePortfolioRisk 等）

### 更新持仓

1. 自动加载最新持仓文件（data/portfolio-YYYY-MM-DD.json）
2. 通过 Agent 调用 MCP 获取最近交易日涨跌幅
3. 自动计算新金额和收益率
4. 支持加仓/减仓操作记录
5. 按交易日期保存（同日多次更新覆盖，历史文件不受影响）

### 自由对话

基金相关问答，Agent 根据 System Prompt 自主选择工具回答。

## 功能范围

| 模块 | 能力 | 状态 |
|------|------|------|
| 持仓分析 | 资产配置、行业集中度、相关性、风险评估、综合诊断 | ✅ |
| 持仓更新 | 涨跌更新、加减仓记录、日期命名文件 | ✅ |
| 基金查询 | 按代码/名称查找，查看详情和业绩 | ✅ |
| 操作策略 | 策略 JSON 注入，分析时自动评估 | ✅ |
| 市场洞察 | 行情解读、热点分析 | 🔜 |
| 策略回测 | 定投模拟、组合回测 | 🔜 |

## 不做

- 实时交易执行
- 非基金品种（股票/债券/期货）
- 用户账户体系
- 移动端 App
