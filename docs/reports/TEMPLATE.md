## 模板说明

### 输出规则

1. **weekly/** - 每周必输出，记录持仓快照
2. **analysis/** - 每周必输出，分析仓位偏离
3. **suggestions/** - 仅触发规则时输出，无触发则注明"本周无需调仓"

### 字段说明

- **较高点**：当前净值相对近期高点（60日）的跌幅
- **补仓触发**：显示触发的档位或距离最近档位的距离
- **止盈触发**：仅在有补仓点记录时计算
- **偏离**：当前占比 - 目标占比

### 补仓点记录

每次实际执行补仓后，必须记录补仓点以便后续止盈计算：
```bash
node -e "require('./dist/state/store').addBuyPoint({
  code: '基金代码',
  date: 'YYYY-MM-DD',
  nav: 补仓时净值,
  amount: 补仓金额
})"
```
